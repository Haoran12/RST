## 增量更新（2026-03-05）- Scene 提取与 Status Panel 在场人物刷新修复

### 后端
1. `SceneService.parse_scene_tag()` 由“多标签取第一条”调整为“多标签取最后一条”，避免同一回复中存在多个 `<scene>` 时读取到旧场景：`backend/app/services/scene_service.py`。
2. 场景人物分隔符扩展为 `,` / `，` / `、` / `;` / `；`，提升 `<scene>` 中 `characters` 解析兼容性：`backend/app/services/scene_service.py`。
3. 更新场景解析单测，覆盖“取最后标签 + 全角冒号 + 混合分隔符”行为：`backend/tests/test_scene_service.py`。

### 前端
1. 聊天成功追加 assistant 消息后，`chat store` 主动广播 `rst-lore-data-changed`（`source=chat-store`），驱动 `StatusPanel` 自动重拉 scene/人物数据，不再依赖手动刷新：`frontend/src/stores/chat.ts`。
